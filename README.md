# Instructions on how to run the code
## Requirements

- **Python**: 3.12 (recommended)
- **pip**: 25.x+

## Setup (proposing virtual environment; run below commands from the project root one by one)
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt

## Run tests
python -m pytest -q

## Run main 
python -m src.main
